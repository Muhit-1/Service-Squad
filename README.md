# Service Squad - Service-Providing Application

Service Squad is a service-providing platform designed to connect users with trusted professionals for a variety of day-to-day services like babysitting, cooking, and home maintenance. This application allows users to book, post, and manage services seamlessly through its user-friendly interface.




## Features

- User Registration and Login
- Job Posting and Job Browsing
- Integration with SQLite Database
- Dynamic UI with JavaFX and Scene Builder
- TilePane for displaying service postings
- Scrollable layouts for efficient UI navigation
- Modular MVC architecture for better scalability


## Tech Stack


Programming Language: Java

UI Framework: JavaFX (with Scene Builder)

Database: SQLite

Build Tool: Maven
